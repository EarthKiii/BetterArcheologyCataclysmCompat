# Required Dependencies

Download the following mod JAR files and place them in this folder:

## 1. Better Archeology Reforged
- **Download from**: https://www.curseforge.com/minecraft/mc-mods/betterarcheology-reforged/files
- **Version**: 1.20.1
- **File**: Look for the latest 1.20.1 Forge version

## 2. L_Ender's Cataclysm
- **Download from**: https://www.curseforge.com/minecraft/mc-mods/l-enders-cataclysm/files
- **Version**: 1.20.1
- **File**: Look for the latest 1.20.1 Forge version

After downloading, place the JAR files in this directory and run:
```
./gradlew build
```

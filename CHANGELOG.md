# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-01-28

### Added
- Initial release
- Compatibility between Better Archeology's Soaring Winds enchantment and Cataclysm's Ignitium Elytra Chestplate
- Single Mixin-based approach for minimal overhead
- Automatic integration with Better Archeology's flight boost mechanics

### Features
- Soaring Winds enchantment can now be applied to Ignitium Elytra Chestplate
- Flight speed boost automatically applies when using enchanted Ignitium Elytra
- Full compatibility with Better Archeology configuration options
- No additional configuration required
